### What I'm working on

**[Clay-Create](https://github.com/blabha/Clay-Create)**
A co-creative ceramic fabrication system using depth camera scanning, height maps, and projection mapping to give participants live digital feedback as they sculpt clay blocks into a continuous wall relief.

**[Talk_To_Tim](https://github.com/blabha/Talk_To_Tim)**
An AI-assisted BIM workflow generating timber structural systems inside Revit, combining a knowledge graph, Eurocode data, and an LLM/GNN pipeline.

**[infrared-hackathon](https://github.com/blabha/infrared-hackathon)**
A climate-aware scheduling tool that scores calendar events for thermal comfort using urban microclimate data. 3rd place, Infrared City Hackathon.

**Other repos:** [Open-to-Sky](https://github.com/blabha/Open-to-Sky) (sky visibility vs. urban heat risk in NYC) · [Disperse](https://github.com/blabha/Disperse) (satellite imagery + deep learning for crowd density) · [CIRCUIT](https://github.com/blabha/CIRCUIT) (CV pipeline for timber upcycling)
